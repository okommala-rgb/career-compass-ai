# Career Compass AI

Career Compass AI is an intelligent career guidance platform designed to help students and aspiring professionals make informed career decisions. By leveraging AI, the application provides personalized recommendations, career insights, and actionable guidance tailored to individual goals and interests.

## Live 

**Application:** https://career-compass-ai.streamlit.app/

---

## Overview

Choosing the right career path can be overwhelming. Career Compass AI simplifies this process by offering data-driven recommendations and interactive guidance to help users understand their strengths, explore opportunities, and plan their next steps with confidence.

Whether you are a student exploring possibilities, a graduate preparing for the job market, or a professional considering a career transition, Career Compass AI serves as your personalized career assistant.

* Personalized career recommendations based on user inputs

## Getting Started

### Prerequisites

* Python 3.10 or later
* OpenAI API Key

### Installation

Clone the repository:

```bash
git clone <repository-url>
cd <repository-name>
```

Create and activate a virtual environment:

```bash
python -m venv .venv
```

Linux/macOS:

```bash
source .venv/bin/activate
```

Windows:

```bash
.venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file and add your API key:

```env
OPENAI_API_KEY=your_api_key_here
```

Run the application:

```bash
streamlit run app.py
```
